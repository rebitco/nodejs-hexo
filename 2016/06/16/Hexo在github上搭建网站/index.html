Hexo在github上搭建网站博客

2016.06.16 king 建站

1.搭建前准备工作

github

git下载: https://git-scm.com/download/

Node.js(4.4.4稳定版)

node.js下载: http://nodejs.cn/

安装环境: win7 64bit

2.环境搭建好后, 接着开始Hexo安装

要用全局安装,加-g参数,打开git-bash.exe

D:\>npm install -g hexo

查看安装好的当前版本

D:\>hexo version

3.Hexo初始化

Hexo安装完毕,接下来要初始化工作,先新建一个用于管理站点的空文件夹,这里我使用D:\blog\

D:\blog\hexo init nodejs-hexo

初始化完成后,会在blog文件夹下生成初始化文件,这时进入初始化文件目录下,并开启Hexo服务器

# 开启hexo服务器
D:\blog\nodejs-hexo>hexo server
[info]Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

通过浏览器打开 http://localhost:4000

到此,表示已部署成功!
